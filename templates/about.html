Feature reference

Settings
  • Search profiles — create multiple named profiles (e.g. "Remote Data Roles", "Portland Only") and toggle between them
  • Job titles — comma-separated; each title becomes a separate search query
  • Locations — comma-separated; the platform loops through every title × location combination
  • Keywords — additional keywords sent to the AI for match scoring (not used in Adzuna queries)
  • Resume upload — paste or upload your resume; enables AI match scoring on every job
  • Find Jobs Now — trigger a scrape without leaving Settings
Jobs tab
  • Scrape Indeed — Playwright-based browser scraper; harder to block
  • Scrape Adzuna — API scraper; fast, reliable, full descriptions included
  • Efficient / Full desc toggle — "Full desc" navigates to each job page for complete descriptions (slower, Indeed only)
  • Match score — 0–100% AI score comparing the job description to your resume; filter by minimum score
  • Show more / show less — expand the description inline without leaving the page
  • View on Indeed / Adzuna — opens the original job listing directly in a new tab
  • View Details — opens the full job page within the platform, including AI match explanation
  • Star — pins the job to the top of the list as a reminder
  • Cover letter — generates a tailored AI cover letter for that specific job
  • Duplicates — jobs with the same external ID are skipped automatically on re-scrape
Applications
  • Status pipeline — Not Applied → Applied → Interview → Offer / Rejected
  • Drag and drop — move jobs between columns on the Kanban board
  • Schedule interview — adds a date/time that appears on the Calendar
  • Notes — freeform notes per application (recruiter name, salary, impressions)
Calendar
  • Interview view — monthly calendar showing all scheduled interviews
  • Upcoming sidebar — the next 5 interviews listed with company and role
  • Click to expand — click a day to see full interview details
Analytics
  • Pipeline funnel — jobs scraped → applied → interviewed → offered
  • More advanced analytics coming soon
AI features
  • Match scoring — runs automatically when jobs are saved; requires resume upload
  • Match explanation — click "View Details" on any job to see why it scored the way it did
  • Cover letter generation — one click on the job card; tailored to that specific posting
  • Re-score — update your resume and re-trigger match analysis from the job detail page

A few tips

  • Keep job titles concise (e.g. "Data Analyst", not "Senior Data Analyst with Python experience") — shorter titles return more Adzuna results.
  • Run Adzuna first — it's faster and already returns full descriptions. Use Indeed for volume.
  • The "Full desc" toggle on the Jobs tab only affects Indeed; Adzuna always returns full descriptions via its API.
  • Re-scrape freely — duplicate jobs are detected and skipped automatically, so you won't get double entries.
  • Upload a detailed resume for better AI scores. The more context, the more accurate the match.
  • Use multiple search profiles in Settings to run different searches for different kinds of roles without losing your current setup.
